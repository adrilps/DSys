A classe EchoServerTCP_Thread_Server funciona como a main do server, a classe ClientController serve como a main do cliente.

A versão do java utilizada é JavaSE-17.

Favor trocar o caminho da variavel MOVIES_FILE_PATH dentro da classe MovieService para o caminho dos filmes dentro do seu computador.

Também trocar USERS_FILE_PATH dentro da classe UsuarioService para o caminho dos usuários dentro do seu computador.
